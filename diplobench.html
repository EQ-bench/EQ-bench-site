About DiploBench

DiploBench tests LLM strategic reasoning and negotiation in the board game Diplomacy. Each model plays as Austria, negotiating with other AI players to survive and win.

For more details and source code, see the GitHub repository.

Note: This is not a benchmark (yet). These are single game runs with high variance between iterations.
